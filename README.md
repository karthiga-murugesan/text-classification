# Text Classification using Machine Learning

This project focuses on classifying text data into predefined categories using machine learning techniques.

## Tools & Technologies
- Python
- Natural Language Processing (NLP)
- Scikit-learn

## Description
Text data was preprocessed using NLP techniques and converted into numerical features using TF-IDF. Machine learning models were trained and evaluated for classification tasks.

## Outcome
The model was evaluated using accuracy and other performance metrics to ensure effective classification of text data.
